Client-side and Server-side Engineering

Client-Side


Client-Side Engineering


  • This refers to everything in a web application that is displayed or takes place on the client (end user device).
  • This includes what the user sees, such as text, images, and the rest of the UI, along with any actions that an application performs within the users browser.
  • These computers request information.
  • Examples of client computers are : mobile phones, laptops, desktops, and tablets.
  • This means that the processing takes place on the users computer. It requires browsers to run the scripts on the client machine without involving any processing on the server.
  • This uses the following scripting languages: CSS, HTML and JavaScript.
  • Its main purpose is to create visual effects and elements for websites, which includes layouts and user interfaces.
  • Gathers input from the users.
  • This is less secure, as users can see and interact with the source code.

Server-Side

Server-side Engineering


  • This means that the processing takes place on a web server. This processing is important as it executes the tasks required by the user on the web.
  • Server-side script is executed in the server thus not making it visible to the users.
  • Uses the following coding languages: PHP, Java, and C#.
  • Focuses more on the more hidden components such as how an application transmits data over to the server.
  • All this does is process the input from the user.
  • These have a higher level of security because the user can not see nor interact with server-side source code.

Differences Between Client-Side and Server-Side Engineering


Client-Side Engineering Server-Side Engineering
~This does not need interaction with the server. ~This does require interaction with the server.
~ Runs on the user's computer. ~ Runs on the web server.
~ Reduces load on the server's prcessing unit ~ It allows the server to provide dynamic websites tailored to the user and increases the processing load on the server.
~ Uses the languages : HTML , CSS and JavaScript. ~ Uses the languages : PHP , ASP.net, C# and Python.
~Code runs on the user's device. ~Code runs through the server.

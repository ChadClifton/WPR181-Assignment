Internet Protocols

IP Definition

“Internet Protocol (IP) is the principal set (or communications protocol) of digital message formats and rules for exchanging messages between computers across a single network or a series of interconnected networks, using the Internet Protocol Suite (often referred to as TCP/IP). Messages are exchanged as datagrams, also known as data packets or just packets.” - Rouse, M. (2023) Internet protocol, Techopedia.

IP is a set of communication instructions that have four layers:

  • Link Layer (lowest)
  • Internet Layer
  • Transport Layer
  • Application layer (highest)

Task of IP

The main task of IP is to transport datagrams from the host source to the destination source, exp: Sending a picture from a phone to a laptop. The IP accomplishes this by putting a “tag” within the datagrams, the tags usually contain address information, and this whole process is called encapsulating.

The steps that an IP takes is like how mail and packages are shipped and delivered. The datagram is encapsulated and sent into the internet by the source host. The datagram holds the “tag” which is the address of the destination source.

“Note: IP is actually a connectionless protocol, meaning that the circuit to the receiver (destination host) does not need be set up before transmission (by the source host). Continuing the analogy, there does not need to be a direct connection between the physical return address on the letter/package and the recipient address before the letter/package is sent.” - Rouse, M.1974

The most complex job of IP is to address and route the datagrams. As Rouse, M states; “However, intelligence in the network is located at nodes (network interconnection points) in the form of routers which forward datagrams to the next known gateway on the route to the final destination.” These routers use what are called IGP, interior gateway protocols, pr EGP, external gateway protocols to help with the routing process. The route is also determined by the prefix in the datagram.

IP Versions

Since IP is the most important protocol of the Internet, many different version of it have been made since its creation in 1974. Currently IP version 4 (IPv4) is the most common and used version, however, version 6 (IPv6) does exist and is gaining popularity across the world.
